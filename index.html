Twitch Link

Twitch라는 방송 플랫폼의 영상 저장 프로그램입니다.

이 프로그램은 Twich라는 인터넷 생방송 플랫폼에서 자체적으로 지원하지 않는 생방송 다운로드, VOD 다운로드드 등과 같은 기능을 제공합니다. C#과 .NET 5를 기반으로 개발되었으며 Twitch의 공개 API를 활용하여 사용자가 원하는 영상을 다운로드합니다.

  • Client: 일반 사용자
  • Category: Utility

Yt-dlp

2006년 개발된 youtube-dl의 비공식 프로젝트로 본 프로젝트의 업데이트 지연을 해결하고자 만들어 졌습니다.

Python 기반 CLI 유틸리티 입니다. Youtube, SoundCloud, Facebook등 다양한 사이트에서 영상 및 오디오 다운로드 기능을 제공합니다. 사용자 UI가 따로 존재하지 않아 Cmd창에서 작업을 수행해야하는 불편함이 있지만 여전히 많은 곳에서 사용되고 있습니다.

  • Client: 일반 사용자
  • Category: Utility

Thunderstore Mod Manager

Overwolf 플랫폼을 기반으로 개발된 데스크탑 어플리케이션 입니다.

C#, .NET 기술 스택을 사용해 개발되었으며 다양한 게임의 모드를 관리하고 설치하는데 사용됩니다. 기본적으로 지원하지 않는 게임 기능이나 사용자의 편의를 위한 MOD들을 설치, 업데이트, 의존성관리 등을 자동으로 처리합니다.

  • Client: Gamers
  • Category: Game Utility
